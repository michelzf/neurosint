# Código de Conduta

## Nosso compromisso

Como participantes, contribuidores e mantenedores deste projeto, comprometemo-nos a fazer
da participação uma experiência livre de assédio para todos — independentemente de idade,
deficiência, etnia, identidade ou expressão de gênero, nível de experiência, nacionalidade,
aparência, raça, religião, ou orientação sexual. Este projeto trata de cuidado e de pessoas
em situação de vulnerabilidade; esperamos um padrão de empatia ainda mais alto.

## Comportamento esperado

- Usar linguagem acolhedora e inclusiva.
- Respeitar pontos de vista e experiências diferentes.
- Aceitar crítica construtiva com elegância.
- Focar no que é melhor para a comunidade e para as famílias que o projeto serve.
- **Proteger a privacidade alheia** — nunca expor dados de saúde reais de ninguém.

## Comportamento inaceitável

- Assédio, insultos, ataques pessoais ou políticos.
- Linguagem ou imagens sexualizadas; atenção sexual indesejada.
- Divulgação de informação privada de terceiros sem permissão (incluindo dados de saúde).
- Qualquer conduta que seria razoavelmente considerada inapropriada num ambiente profissional.

## Aplicação

Casos de comportamento abusivo podem ser reportados **em privado** ao mantenedor por um destes
canais: uma mensagem direta no GitHub para **[@michelzf](https://github.com/michelzf)**, ou um
relatório privado pela aba **Security → Report a vulnerability** deste repositório.
Todas as queixas serão analisadas e investigadas, resultando em uma resposta apropriada às
circunstâncias. Mantenedores que não seguirem ou aplicarem este código de boa-fé podem
sofrer repercussões temporárias ou permanentes.
